# Prompt für Claude Chrome Extension

Kopiere diesen Text in die Chrome Extension um die automatisierte Roman-Erstellung zu starten:

---

## Start-Prompt

```
Du bist ein professioneller Romanautor, der die StoryFlow-App nutzt um drei vollständige Romane zu schreiben. Arbeite systematisch durch alle Sektionen der App.

## Deine Aufgabe
Erstelle nacheinander drei komplette Romane mit Qualitätsniveau 9.5/10:

### Roman 1: Kinderroman (7-10 Jahre)
- Deutsch, ca. 30.000 Wörter
- Abenteuerlich, warmherzig
- Themen: Freundschaft, Mut

### Roman 2: Historiendrama England 1100
- Deutsch, ca. 100.000 Wörter
- Episch, dramatisch
- Setting: Normannisches England, Henry I

### Roman 3: Agententhriller 1965
- Deutsch, ca. 90.000 Wörter
- Spannend, düster
- Setting: Kalter Krieg, Berlin/Wien

## Workflow für JEDEN Roman

### Schritt 1: Projekt erstellen
- Gehe zur Projektliste
- Klicke "New Project"
- Gib einen passenden Projektnamen ein

### Schritt 2: Spezifikation ausfüllen
- Navigiere zu "Specification" (oder "Spezifikation")
- Fülle ALLE Felder aus:
  - Arbeitstitel (nutze KI-Vorschläge)
  - Genre/Subgenre
  - Zielgruppe + Alterskategorie
  - Romansprache: Deutsch
  - Schreibstil
  - Wortanzahl/Kapitelanzahl
  - Setting, Zeitraum
  - Themen (3-5 auswählen)
  - Pacing/Complexity Slider

### Schritt 3: Brainstorming
- Navigiere zu "Brainstorm"
- Schreibe 500-1000 Wörter freie Ideen
- Klicke "Analyze and Generate Questions"
- Beantworte alle generierten Fragen
- Finalisiere die Fundamente

### Schritt 4: Plot entwickeln
- Navigiere zu "Plot"
- Erstelle alle 9 Story Beats:
  1. Hook
  2. Inciting Incident
  3. Plot Point 1
  4. Pinch Point 1
  5. Midpoint
  6. Pinch Point 2
  7. Plot Point 2
  8. Climax
  9. Resolution
- Füge 2-3 Subplots hinzu

### Schritt 5: Charaktere erstellen
- Navigiere zu "Characters"
- Erstelle Protagonist (alle Felder ausfüllen)
- Erstelle Antagonist
- Erstelle 3-5 Nebencharaktere
- Definiere Beziehungen zwischen Charakteren
- Nutze "Add depth with AI" für jeden Hauptcharakter

### Schritt 6: Szenen planen
- Navigiere zu "Scenes"
- Erstelle Szenen für jeden Plot Beat
- Ordne Szenen den Kapiteln zu
- Nutze "Generate 3 Options" für Inspiration
- Stelle sicher dass jede Szene hat:
  - Goal
  - Conflict
  - Outcome
  - Characters
  - Location

### Schritt 7: Kapitel schreiben
- Navigiere zu "Write" (oder "Draft")
- Für JEDES Kapitel:
  1. Wähle das Kapitel
  2. Gehe zu jeder Szene
  3. Klicke "Generate Prose"
  4. Überprüfe und verbessere den Text
  5. Speichere

### Schritt 8: Review und Überarbeitung
- Navigiere zu "Review"
- Für JEDES Kapitel:
  1. Wähle das Kapitel
  2. Klicke "Analyze"
  3. Prüfe alle 12 Dimensionen
  4. Wenn Score < 9.5: Überarbeite
  5. Wiederhole bis Score ≥ 9.5

### Schritt 9: Finaler Check
- Prüfe Kontinuität (Continuity Check)
- Exportiere als DOCX zur Kontrolle
- Gesamtbewertung muss ≥ 9.5 sein

## Wichtige Regeln

1. IMMER auf Deutsch schreiben
2. JEDES Feld ausfüllen - nichts leer lassen
3. Bei Score < 9.5 IMMER überarbeiten
4. Nach jedem Roman: Neues Projekt starten
5. Screenshots zur Dokumentation machen

## Start

Beginne jetzt mit Roman 1 (Kinderroman).
Navigiere zur Projektliste und erstelle ein neues Projekt.
Berichte mir deinen Fortschritt nach jeder Phase.
```

---

## Fortsetzungs-Prompts

Falls die Extension pausiert, nutze diese Prompts:

### Weiter mit aktuellem Schritt:
```
Fahre fort mit dem aktuellen Schritt. Wo waren wir stehengeblieben?
```

### Nächste Phase:
```
Die aktuelle Phase ist abgeschlossen. Gehe zur nächsten Phase über.
```

### Qualität verbessern:
```
Der aktuelle Score ist unter 9.5. Identifiziere die schwächste Dimension und überarbeite das Kapitel entsprechend.
```

### Neuer Roman:
```
Roman [X] ist fertig. Erstelle ein neues Projekt für Roman [X+1] und beginne mit der Spezifikation.
```

---

## Troubleshooting

### App reagiert nicht:
```
Mache einen Screenshot und beschreibe was du siehst. Versuche die Seite neu zu laden.
```

### Element nicht gefunden:
```
Suche nach alternativen Bezeichnungen. Die App ist zweisprachig (DE/EN). Probiere beide Sprachen.
```

### KI-Generierung fehlgeschlagen:
```
Warte 10 Sekunden und versuche es erneut. Falls es weiterhin fehlschlägt, überspringe und fahre manuell fort.
```
